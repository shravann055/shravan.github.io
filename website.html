Several attributes and elements can enhance dropdown lists.
The selected attribute makes an option the default choice,
while disabled can disable the entire dropdown or individual options.
The multiple attribute allows selecting multiple options,
and required mandates selection before form submission.
The optgroup element groups related options under a label.
For more complex dropdowns, CSS, JavaScript,
or frameworks like Bootstrap are often used.
A basic example of a dropdown list and grouping options can be found on Mimo.
